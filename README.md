# Dealership Review Application

## Project Description

This is a full-stack dealership review application developed as part of the IBM Full Stack Software Developer Capstone.

## Technologies Used

- React
- Django
- Python
- Express.js
- MongoDB
- GitHub Actions
- Docker

## Features

- User Registration
- User Login & Logout
- Dealer Listing
- Dealer Details
- Add Review
- Sentiment Analysis
- CI/CD Pipeline
- Deployment
